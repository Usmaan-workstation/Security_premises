# Enterprise Open-Source SIEM Pipeline

> Enterprise Security Monitoring Architecture using Open-Source Technologies

## Overview

Enterprise Open-Source SIEM Pipeline is a security monitoring project demonstrating how enterprise-grade Security Operations Center (SOC) capabilities can be implemented using open-source technologies.

The objective was to design a complete detection pipeline capable of transforming raw infrastructure telemetry into actionable security alerts aligned with the MITRE ATT&CK framework.

The project focuses on practical SOC engineering concepts including log ingestion, normalization, detection engineering, alert correlation, incident workflows, and case management.

---

## Objectives

* Centralize security telemetry
* Normalize heterogeneous log sources
* Develop detection logic
* Map detections to MITRE ATT&CK
* Improve alert quality
* Build an end-to-end SOC workflow
* Demonstrate enterprise SIEM architecture
